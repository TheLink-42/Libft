## About the project

This library recreates of the standard C librry functions and introduce additional utility functions.

### Allowed external functions

<details>
<summary>Functions</summary>

- malloc <stdlib.h>
- free <stdlib.h>
- write <unistd.h>
</details>

### Hand-in functions

<details>
<summary>Mandatory</summary>

- atoi					
- bzero
- isalnum				
- isalpha
- isascii				
- isdigit
- isprint				
- memchr
- memcmp				
- memcpy
- memmove				
- memset
- strchr				
- strlcat
- strlcpy				
- strlen
- strncmp				
- strnstr
- strrchr				
- tolower
- toupper


- strdup
- calloc



Aditional functions:


- ft_substr
- ft_strjoin
- ft_strtrim
- ft_split
- ft_itoa
- ft_strmapi
- ft_striteri
- ft_putchar_fd
- ft_putstr_fd
- ft_putendl_fd
- ft_putnbr_fd
</details>

<details>
<summary>Bonus</summary>

As a bonus, it creates some new functions based on a struct to work as linked lists.

- ft_lstnew
- ft_lstadd_front
- ft_lstadd_back
- ft_lstsize
- ft_lstlast
- ft_lstdelone
- ft_lstclear
- ft_lstiter
- ft_lstmap
</details>
